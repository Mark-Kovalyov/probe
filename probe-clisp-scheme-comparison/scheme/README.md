# Scheme string functions

## Check atom for string
```
> (string? "Hello")

;Value: #t

> (string? 1)

;Value: #f

```

## String to list
```
> (string->list "Hello")

;Value: (#\H #\e #\l #\l #\o)

```

## Substring

```
(substring "Hoola-hoola" 2 5)        

;Value: "ola"
```
